Sight Assit es una extension de Navegador que consiste de un menú que se compone de 7 opciones diferentes:
- Invertir Colores: Invierte los colores de la pagina, lo que le permite al usuario ajustar el tono de la pagina a uno que se ajuste mejor a su sensibilidad al contraste.
- Tamaño del Texto: Cada vez que se hace clic en esta opción, cambia el texto de la pagina hasta 3 tamaños diferentes, cuando alcanza el tamaño máximo, el texto regresa a su tamaño original.
- Tamaño del Cursor: Aumenta el tamaño del cursor, haciéndolo mas fácil de ubicarlo.
- Línea de Lectura: Activa una línea que se mantiene adherido al cursor hasta que esta opción sea desactivada, esta función fue pensada para que no se pierda de vista la posición del cursor
- Subrayar Texto: Subraya el texto que se haya seleccionado previamente al dar clic en la opción.
- Lectura de voz: En caso de que al usuario se tenga dificultades para leer, el usuario dispone de esta opción para activar una lectura automática del texto seleccionado.
- Reiniciar Cambios: Desactiva todos los cambios de las opciones anteriores que se hayan mantenido activas.
